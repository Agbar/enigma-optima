NAME
        fgstart  -  start enigma-client in the foreground


SYNOPSIS  
        fgstart


DESCRIPTION
        This batch file is generated by the installer. You'll find it
        in the run directory created by the installer. If you have not
        automated client startup, you can use it to start the client in
        the foreground.


SEE ALSO
        enigma-client(1), eclient-rpc(1)
